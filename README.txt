The easiest way to add code to the header. Great for Google Analytics and more! Supports scripts, css, or any valid HTML.

**Supported Languages**

* English

== Installation ==

Install Better Header Code the same you would any other plugin!

Dashboard Method:

1. Login to your WordPress admin and go to Plugins -> Add New
2. Type "Better Search Replace" in the search bar and select this plugin
3. Click "Install", and then "Activate Plugin"


Upload Method:

1. Unzip the plugin and upload the "better-header-code" folder to your 'wp-content/plugins' directory
2. Activate the plugin through the Plugins menu in WordPress

== Frequently Asked Questions ==

= Using Better Header Code =

Once activated, Better Header Code will add a page under the "Settings" menu page in your WordPress admin.

= Is my host supported? =

Yes! This plugin should be compatible with any host.

= Can I damage my site with this plugin? =

Not permanently! If you save bad code, you can easily remove it.
